# Naming Conventions

## Files & Folders
- **Folders:** kebab-case (e.g., `user-profile`)
- **React Components:** PascalCase (e.g., `Button.tsx`)
- **React Hooks:** camelCase (e.g., `useAuth.ts`)
- **Services/Providers:** PascalCase (e.g., `AuthService.ts`)
- **Utility Files:** kebab-case or camelCase (e.g., `date-utils.ts`)
- **TypeScript Types/Interfaces:** PascalCase (e.g., `UserType.ts`)

## Code
- **Variables/Functions:** camelCase
- **Constants:** SCREAMING_SNAKE_CASE
- **Types/Interfaces:** PascalCase (prefix `I` for interfaces if needed, but preferably not)
- **Classes:** PascalCase
- **CSS Modules:** kebab-case
